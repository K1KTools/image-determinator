English:

K1K Image Determinator is a highly optimized, browser-based 2D physics simulator designed to create stunning particle-reveal animations. It calculates the interactions of thousands of rigid bodies, mapping a custom image onto the settled particles. Built for absolute performance, it features invisible background frame-baking to max out CPU usage, custom .json project import/export, and deep physics customization (including sub-steps and collision iterations). With this tool, you can easily bake, preview, and export fluid .webm animations entirely client-side.

Español:

K1K Image Determinator es un simulador físico 2D de alto rendimiento para el navegador, diseñado para crear impresionantes animaciones de partículas que revelan imágenes. Calcula la interacción de miles de cuerpos rígidos, mapeando una imagen personalizada sobre las partículas una vez asentadas. Desarrollado con el foco en la optimización, cuenta con renderizado invisible en segundo plano para aprovechar al máximo la CPU, importación/exportación de proyectos en formato .json nativo, y personalización avanzada de físicas (incluyendo sub-pasos e iteraciones de colisión). Con esta herramienta, podés hornear, previsualizar y exportar animaciones fluidas en .webm de forma enteramente local.
